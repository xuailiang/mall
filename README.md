# 京东风格商城前端（Vue3 + Vite）

这是一个移动端电商风格的 Vue3 脚手架，默认集成 NutUI，并内置京东风格设计 Token 与页面模板。

## 运行

1. 安装依赖
   ```bash
   npm install
   ```

2. 启动开发服务器
   ```bash
   npm run dev
   ```
